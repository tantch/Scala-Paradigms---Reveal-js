Scala's History

Where does it come from?

Scala, a general-purpose, object-oriented, functional language for the JVM

created by Martin Odersky

Scala comes from java and has full support for functional programming and a strong static type system

Scala was design to support many of the criticised shortcommings of JAVA

		println("Hello, World!")

Scala's Advantages

Compared to Java

  • Reduced code sizes
  • More concise and clear than Java
  • Use all the java libraries and all the advantages of running on a JVM
  • Closures and functions as part of the language
  • Operator overloading
  • And much more ...

Scala: 

						def mathFunction(num: Int): Int = {
  var numSquare: Int = num*num
  return (math.cbrt(numSquare) + math.log(numSquare)).
    asInstanceOf[Int]
}
