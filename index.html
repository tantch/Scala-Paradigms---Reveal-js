Scala's History

Where does it come from?

Scala, a general-purpose, object-oriented, functional language for the JVM

created by Martin Odersky

Scala comes from java and has full support for functional programming and a strong static type system

Scala was design to support many of the criticised shortcommings of JAVA

		println("Hello, World!")

Scala's Advantages

Compared to Java

  • Reduced code sizes
  • More concise and clear than Java
  • Use all the java libraries and all the advantages of running on a JVM
  • Closures and functions as part of the language
  • Operator overloading
  • And much more ...

	    if (addToGrid)
	      grid.add(this);
	  }

	    this(0.0, 0.0);
	  }

	  public double getX() {
	    return x;
	  }

	  public double getY() {
	    return y;
	  }

	  double distanceToPoint(final Point other) {
	    return distanceBetweenPoints(x, y,

	  }

	  private static Grid grid = new Grid();

	  static double distanceBetweenPoints(

Scala: 

						class Point(
	    val x: Double, val y: Double,
	    addToGrid: Boolean = false
	) {
	  import Point._

	  if (addToGrid)
	    grid.add(this)

	  def this() = this(0.0, 0.0)

	  def distanceToPoint(other: Point) =
	    distanceBetweenPoints(x, y, other.x, other.y)
	}
